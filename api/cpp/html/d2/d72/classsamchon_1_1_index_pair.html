Detailed Description

template<class _Ty>
class samchon::IndexPair< _Ty >

A pair of index and its value(_Ty)

IndexPair is a std::pair<size_t, _Ty> class.

IndexPair expresses index and value. It's useful for expressing finding literals, specified object, calculating mininum or maximum with its position.

[Inherited] This class couples together a pair of values, which may be of different types (T1 and T2). The individual values can be accessed through its public members first and second.

Pairs are a particular case of tuple.
