Implements IDictionary

Invoke class is a container set of InvokeParameter and supports conversion to Invoke Message The Invoke Message is a standard messsage for Network I/O in Samchon Framework It's based on XML and have the structure like the example Example message: <invoke listener="{$FUNC_NAME}">      <parameter name="{$PARAM_NAME}" type="{$PARAM_TYPE}" >{$VALUE}</parameter>      <parameter name="{$PARAM_NAME}" type="{$PARAM_TYPE}" >{$VALUE}</parameter>      ... </invoke> <invoke /> -> class Invoke <parameter /> -> class InvokeParameter
